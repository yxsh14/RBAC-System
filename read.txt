Admin Routes 

1. Create Blog -POST

Header: Authorization: Bearer <TOKEN> (admin user checked)

Body (JSON):
	{
    "title":"Blog1",
    "content":"This is the blog created by admin yash."
	}


2. Edit Blog -PATCH
URL: http://localhost:8001/api/edit/:BlogId
Header: Authorization: Bearer <TOKEN> (admin user Checked)

	{
    "title":"Blog1",
    "content":"This is the blog edited by admin yash."
	}
